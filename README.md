# Página de cotización HAZTHINK

Página estática y autocontenida para presentar la cotización COT-HAZTHINK-2026-001.

## Abrir

Abrir `index.html` en navegador.

## Publicar

Subir esta carpeta completa a cualquier hosting estático. Mantener juntos:

- `index.html`
- `styles.css`
- `app.js`
- `assets/`

El botón para descargar PDF usa la ruta:

`../Cotizacion_HAZTHINK_5000_MXN.pdf`

Si la página se publica sin carpeta superior, copiar también el PDF y actualizar esa ruta.
